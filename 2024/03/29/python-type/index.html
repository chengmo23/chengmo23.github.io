可变与不可变类型

  • 可变类型
    值改变的情况下，id 不变，说明改的是原值（也就是内存地址不变，变量还是指向原来的内存地址）
  • 不可变类型
    值改变的情况下，id 变了（也就是申请了新的内存地址，变量指向了新的内存地址）

补充

字典类型数据：{key: value} 中的key必须为不可变类型，但一般均为str类型
