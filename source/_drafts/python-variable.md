## 变量的命名规则

- 变量名的组成必须是字母、数字、下划线
- 不能以数字开头
- 不能用 `python` 关键字命名

## 变量的 type 和 id
- **type** 表示变量的类型
- **id** 为根据变量值内存地址计算出的一个 `id` 号码

## 变量的比较
- **==** 比较两个变量的 `值` 是否相等
- **is** 比较两个变量的 `id` 是否相等

## 小整数池
```python
[-5, 256]
```

## 大整数池
- 只存储包含标准字符(数字、字母、下划线)的字符串，包含特殊字符串不会被添加到大整数池中

```python
'''例子：'''
a = "123"
b = "123"
# a is b = True
# 这两个指向的内存地址是一样的

c= "123#"
d = "123#"
# c is d = False
# 这两个指向的内存地址是不一样的
```

## 常量
> python语法中没有常量的概念，
> 但是在程序开发过程中涉及到常量的概念，
> 在python中，约定或者说规范全大写的变量表示常量

```python
PIE = 3.141592653589
AGE = 120
```
